Problemas com a API de busca do Twitter

05.Aug.2021

Hoje alterei o @DevBotBR para buscar termos relacionados a TI além da hashtag #devbr.

Tive problemas para buscar o termo "php" pela API do Twitter por conta da extensão ser comum em links, por isso resolvi deixar o "php" de fora da busca do bot.

Outros problemas recorrentes são o retorno de tweets resumidos (excerpts) ao invés da versão integral, links com encurtadores de url quebrados e o fato da api buscar o termo no username dos perfis (ex: fullstack, sql, etc).

Para não exceder o limite de buscas coloquei o script para rodar a cada 15 minutos.
